# Fmusic-移动手势音乐播放器

#### 项目介绍
一款移动端的手势音乐播放器，支持单曲循环，歌词显示，歌曲搜索，音量控制，列表循环等功能
采用Sition.js打造，项目地址：https://gitee.com/lovefc/Sition
